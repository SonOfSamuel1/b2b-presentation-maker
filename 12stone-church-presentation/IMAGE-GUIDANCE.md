# 12Stone Church Presentation - Image Guidance

This document provides detailed guidance on what types of photos to source for each slide in the presentation.

---

## Overview

- **Total Images Needed**: 16 photos across 6 slides
- **General Style**: Professional, high-quality, authentic church photography
- **Aspect Ratios**: Mix of landscape (16:9, 3:2) and portrait orientations
- **Resolution**: Minimum 1920px on longest side for crisp display
- **File Format**: JPG or PNG

## General Photography Guidelines

### Technical Requirements
- **Resolution**: Minimum 1920px on longest side
- **File Format**: JPG (optimized) or PNG
- **Color Profile**: sRGB for web display
- **Compression**: Moderate (balance quality and file size)

### Visual Style
- **Lighting**: Well-lit, professional quality
- **Composition**: Clean, uncluttered backgrounds
- **Color**: Consistent with brand palette (#A7D4F0 accent blue)
- **Authenticity**: Real people from 12Stone community preferred over stock
- **Diversity**: Show range of ages, families, demographics

### Legal Considerations
- Obtain photo release forms for identifiable individuals
- Verify licensing for stock photos if used
- Credit photographers appropriately in final materials
- Ensure partnership organizations approve use of mission photos

## Implementation Instructions

Once you have sourced all images:

1. **Save images** to the folder structure above
2. **Edit each HTML file** to uncomment the `<img>` tag and update the `src` path
3. **Remove or comment out** the `.image-placeholder` div
4. **Test rendering** by opening HTML files in browser
5. **Re-export** presentation using `node generate-12stone.js`

Example edit in `slide-01-hero.html`:
